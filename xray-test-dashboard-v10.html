📡 API Endpoints Reference
Jira REST API v3 — Find Test Executions
GET /rest/api/3/search?jql=issuetype="Test Execution" AND fixVersion="{version}"&fields=summary,status,assignee,fixVersions,labels,environment,customfield_*
Xray Server/DC — Test Runs for Execution (with pagination)
GET /rest/raven/2.0/api/testexec/{execKey}/test?detailed=true&start=0&limit=100
Xray Cloud — GraphQL Test Runs
POST https://xray.cloud.getxray.app/api/v2/graphql
Xray Cloud — Authenticate
POST https://xray.cloud.getxray.app/api/v1/authenticate
